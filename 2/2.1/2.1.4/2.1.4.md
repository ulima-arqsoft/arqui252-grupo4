# 2.1.4. Módulo 4 - Sistemas de Pago
# Sistema de Pagos

| **ID** | **Nombre del Requisito** | **Historia de Usuario** |
|--------|----------------------------|---------------------------|
| RF-01 | Procesamiento seguro de pagos | Como comprador, quiero realizar pagos seguros a través de la plataforma de Culqi para completar mis compras sin riesgos. |
| RF-02 | Integración con pasarelas de pago | Como usuario, quiero poder pagar con tarjetas, Yape o Plin a través de la integración única con Culqi. |
| RF-03 | Validación de transacciones | Como sistema, quiero recibir una confirmación (Webhook) de Culqi para verificar el éxito o fallo del pago. |
| RF-04 | Emisión de comprobante digital | Como comprador, quiero recibir un comprobante digital de mi compra una vez que el pago sea confirmado. |
| RF-05 | Registro de transacciones | Como administrador, quiero registrar todas las transacciones confirmadas por Culqi para control contable. |
| RF-06 | Protección de datos financieros | Como usuario, quiero que mis datos financieros sean manejados directamente por Culqi para evitar fraudes. |
